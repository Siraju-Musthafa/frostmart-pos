# FrostMart POS

A full-stack Point of Sale (POS) and Inventory Management System designed for frozen food stores. Built with React, Node.js, Express, PostgreSQL, Prisma ORM, and Tailwind CSS.

## Features

- User Authentication (JWT)
- Role-Based Access Control
- Product & Category Management
- Customer Management
- POS Billing System
- Sales History & Invoice Generation
- Supplier Management
- Purchase Tracking
- Customer Ledger System
- Dashboard Analytics
- Responsive Mobile-Friendly UI
- PDF Invoice Download & Printing
